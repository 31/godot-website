learn godot

Start learning godot.

Godot development is open. This means you can fix or improve any part of the engine yourself and choose whether to contribute it back to keep it private. New features are always available to use and test, without the need of having to wait for the next major release. Compiling Godot from source is very easy (it has no common dependencies) and the process (for each platform) is well-documented on the wiki. To obtain the source code (and consulting the documentation), please visit the GitHub project page.
